This is the instruction of using this program:
1. Download and install XAMPP
2. Open phpmyadmin and import 'adprog.sql'. Make sure the name of the database is 'adprog'
3. Link the 'mysql-connector-java-5.1.18-bin' jar  file to the eclipse
4. Open the 'project' package in eclipse
5. Run the 'main.java' program
