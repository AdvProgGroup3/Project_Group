This is the instruction of using this check in program:
1. Download and install XAMPP
2. Open phpmyadmin and import 'adprog.sql'. Make sure the name of the database is 'adprog'
3. Link the 'mysql-connector-java-5.1.18-bin' jar  file to the eclipse
4. Open the 'project' package in eclipse
5. Run the 'GetLocation.java' program
6. input the user id. It must be in integer number (In database there are only 1,2 ,5,6 and 7 user id input. Which 1 and 2 does not booked yet. and 5,6 7 already booked)
